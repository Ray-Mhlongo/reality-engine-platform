Build a 30-second mockup ad that clearly demonstrates how the app is used.

## 30s Timeline (Exact Template Order + Timing)

### Scene 1 (0:00-0:03) - Hook
- Template: `iPhone 14`
- Media: Showcase hero screenshot
- Text: `Too many customers. Too little time.`
- Motion: subtle zoom-in

### Scene 2 (0:03-0:07) - Start sale
- Template: `iPhone 12 Real`

- Template: `iPhone 13`
- Media: Checkout screen (product + qty + payment)

### Scene 4 (0:12-0:17) - Loyalty
- Template: `iPad Mini 6`
- Media: Loyalty tracker screen
- Text: `Loyalty that brings customers back`

### Scene 5 (0:17-0:22) - Insights
- Template: `MacBook Pro M1 14`

- Text: `Know your numbers daily`

### Scene 6 (0:22-0:26) - Promotion engine
- Template: `Google Pixel 6`
- Media: Broadcast screen
- Text: `Send smart WhatsApp promos`

### Scene 7 (0:26-0:30) - End card
- Template: `iPhone 14`
- Media: Showcase screen or clean brand card
- Text line 1: `Cathdel Creamy`
- Text line 2: `Sell Faster. Reward More. Grow Daily Revenue.`
- Add small badge/logo: `Powered by Data Insights by Ray`

## Build Steps in Previewed
1. Open `https://previewed.app/editor/3D/`.
2. Create 7 scenes.
3. Apply templates in the timeline order above.
4. Drop each screenshot into its matching device scene.
5. Apply text overlays exactly as listed.
6. Keep each scene duration locked to the timeline above.
7. Export `MP4` at `1080x1920` for Reels/TikTok.
8. Optional second export: `1920x1080` for YouTube/Facebook.

## Timing Map (Quick Reference)
- `0-3`: Problem
- `3-7`: Sell start
- `7-12`: Checkout
- `12-17`: Loyalty
- `17-22`: Dashboard
- `22-26`: Broadcast
- `26-30`: Brand CTA

## Audio + Captions
- Keep music upbeat but clean.
- Keep captions high-contrast (white with dark shadow).
- Leave 0.2-0.4s breathing space before final CTA.
