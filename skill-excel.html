Analysis and dashboard stage

Excel

I used Excel to analyze and explore the structured data exported from the SQL database. It allowed me to work with customer subscription ticket and outage data in a flexible environment where I could build calculations and summaries. This helped me identify patterns and trends across different parts of the business.

I also used Excel to create dashboards that present insights in a visual and easy to understand format. These dashboards show key performance indicators such as customer growth subscription status and support activity. This step demonstrates how raw data can be transformed into meaningful business insights.

Excel represents the analysis layer of this project where structured data becomes actionable information. It connects the raw database work from SQL with the visual and interpretive stages of analysis.
